# 客户与项目管理系统（单文件离线版）

这是一个**无需安装依赖**的本地网页工具。

## 使用方法

1. 直接双击 `index.html`
2. 在浏览器中打开后即可使用
3. 数据自动保存到浏览器 `localStorage`

## 功能

- 客户管理：新增、编辑、删除、搜索
- 项目管理：新增、编辑、删除、搜索、按客户/状态/优先级筛选、智能排序
- 项目列表紧凑模式 + 分页（10/20/50/100、上一页/下一页、总数与页码）
- 项目状态流转：进行中 / 待跟进 / 已完成 / 已暂停 / 已归档
- 一键“标记完成”（自动记录完成日期）
- 项目快速复用：复制旧项目为新项目、选客户后显示最近项目参考
- 客户新增支持“套用已有客户资料模板”
- 跟进提醒（今天/本周/逾期）
- 质量问题记录模块
- 样机发货记录模块
- 首页行动看板（今天要跟进、本周要跟进、逾期、最近更新）
- Excel 数据导出/导入（主功能，含：客户/项目/跟进记录/质量问题/样机发货 多 Sheet，可改后再导回）
- JSON 数据导入/导出（备用备份）
