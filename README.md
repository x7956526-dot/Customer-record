# 客户与项目管理系统（单文件离线版）

这是一个**无需安装依赖**的本地网页工具。

## 使用方法

1. 直接双击 `index.html`
2. 在浏览器中打开后即可使用
3. 数据自动保存到浏览器 `localStorage`

## 功能

- 客户管理：新增、编辑、删除、搜索
- 项目管理：新增、编辑、删除、搜索、按阶段筛选
- 客户与项目关联
- 项目跟进记录（新增、删除、倒序）
- 跟进提醒（下次跟进日期、跟进事项、完成标记、首页今日/逾期提醒）
- 质量问题记录模块
- 样机发货记录模块
- 首页看板统计
- JSON 数据导入/导出
- Excel 导入（按固定列名：客户、开始时间、完成时间、内容、计划安排、责任人、备注）
